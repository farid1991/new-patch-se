# NEW-PATCH-SE

## Sony Ericsson Patch
Patch sources for Sony Ericsson phones A1 and A2 platform.

Build with CMake and ARM-GCC

### alarm_remaining_time
```
Show remaining time before the alarm.
Displayed on the first line in the list item and alarm clocks on your standby.
Display format "hh:mm (hh:mm)".
```

### book_manager
```
Improved Activity Menu.
1st tab corrected for the title "New Events: %d".
2nd tab displays running books GUI + header "Heap: %dKB".
3rd tab displays running elf + in the title count of the elves in the memory.
4th tab: shut down, restart, brightness, flight mode, close the connection.
```

### bt_transmission
```
Displays a process of transmission through BT in the status bar.
Only works if the window "Sending"/"Receiving" is minimazed.
```

### db_itemtext
```
Display more info of file and/or folder on FileManager when view mode set to Thumbnail List.
File: Show file size on second line of list item.
Folder: Show count of files and/or sub-folder inside it on second line of list item.
```

### db_pause
```
Add pause button during audio playback through the File Manager.
```

### db_player
```
Add advanced display information when playing music on File Manager.
```

### exif_info
```
Display Exif Info of JPEG file.  
Added 40 Exif tags on JPEG file information.  
```

### flashmenu_picker
```
Ability to select SWF as Custom Flash Menu.
Select swf from Main Menu Layout->Custom->FlashMenuPicker.
```

### fm_record
```
Add "Record" functions in the FM Radio.
```

### mp3_info
```
Add extra informations about files mp3.
Added items Bitrate, Samplerate and Channel mode.
```

### phone_wo_sim
```
Fully operational phone without a SIM-card.
Everything works except the functions that require cellular communication.
v1.2
```

### phonebook_title
```
Display the total of phone and sim contacts in the second row of the header.
```

### png_info
```
Add more info about PNG in file information window.
```

### remove_scrollbar
```
Remove Scroll bar on Main Menu.
```

### replace_file_bt
```
Add question about replacing file if exist when receiving file via Bluetooth.
```

### shutdown_menu
```
Add Power Menu when long press End Button in standby.
```

### sleepmode
```
Change the items in a sleep mode.
```

### volume_redraw
```
Ability to redraw VolumeControl with custom skin.
Use VolumeSkinEditor.elf to edit and set skin.
```

### volume_slider
```
Add New Volume Control GUI on settings.
```
