# NEW-PATCH-SE

## Sony Ericsson Patch
Patch sources for Sony Ericsson phones A1 and A2 platform.

Build with CMake and ARM-GCC

### adv_player
```
AdvPlayer
Adds possibility to edit the Position,Color,Font
Enable/Disable Title,Artist,Album,Total/Elapsed/Remaining time,Sample rate,Total/Current track number,progress-bar,Album art
Adds possibility to read Album Art from MP3
Adds new item of Sample rate to Walkman
Adds possibility to edit position of icons shuffle/EQ/loop/state,Title/Artist/Album/Background
Adds possibility to use skins in Walkman Player
Adds possibility to display Walkman in Fullscreen
Adds possibility to display visualization Walkman
Adds Album in TopLevelBook menu
Using theme color in TopLevelBook Menu
Redraws everything in Walkman,optimized for better Walkman experience
```

### alarm_remaining_time
```
Show remaining time before the alarm.
Displayed on the first line in the list item and alarm clocks on your standby.
Display format "hh:mm (hh:mm)".
```

### book_manager
```
Improved Activity Menu.
1st tab corrected for the title "New Events: %d".
2nd tab displays running books GUI + header "Heap: %dKB".
3rd tab displays running elf + in the title count of the elves in the memory.
4th tab: shut down, restart, brightness, flight mode, close the connection.
```

### brightness
```
Extended brightness adjustment range.
Allows you to adjust brightness from 0 to 100%.
```

### bt_transmission
```
Displays a process of transmission through BT in the status bar.
Only works if the window "Sending"/"Receiving" is minimazed.
```

### db_itemtext
```
Display more info of file and/or folder on FileManager when view mode set to Thumbnail List.
File: Show file size on second line of list item.
Folder: Show count of files and/or sub-folder inside it on second line of list item.
```

### db_pause
```
Add pause button during audio playback through the File Manager.
```

### db_player (A2)

```
Add advanced display information when playing music on File Manager.
```

### db_redraw (A1)

```
Add advanced display information when playing music on File Manager.
```

```
Change Softkey in Standby to "Goto" Shortcut.
```
